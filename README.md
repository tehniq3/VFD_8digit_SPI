# VFD_8digit_SPI
based on info and tests from https://github.com/tehniq3/VFD_SPI

Display        Wemos D1 Mini (ESP8266)
--------------------------------------
- GND  --------- GND
- VCC  --------- 5V
- EN   --------- 3V3
- RESET -------- D1 (GPIO5)
- CS   --------- D8 (GPIO15)
- CLK  --------- D5 (GPIO13)
- DIN  --------- D7 (GPIO14)
